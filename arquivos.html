Comandos Básicos do Linux

  • ls - Lista arquivos e diretórios
  • cd - Navega entre diretórios
  • pwd - Mostra o diretório atual
  • mkdir - Cria um novo diretório
  • rm - Remove arquivos ou diretórios
  • cp - Copia arquivos ou diretórios
  • mv - Move ou renomeia arquivos ou diretórios
  • cat - Exibe o conteúdo de arquivos
  • echo - Exibe mensagens ou redireciona saídas
  • man - Acessa o manual de comandos
  • touch - Cria arquivos vazios ou atualiza a data de modificação
  • find - Localiza arquivos e diretórios
  • grep - Busca texto em arquivos
  • chmod - Altera permissões de arquivos
  • chown - Altera o proprietário de arquivos
  • tar - Compacta e descompacta arquivos
  • zip - Compacta arquivos em formato ZIP
  • unzip - Descompacta arquivos ZIP
  • ps - Exibe processos em execução
  • kill - Encerra processos
  • top - Monitora processos em tempo real
  • df - Exibe o uso de espaço em disco
  • du - Exibe o uso de espaço por arquivo/pasta
  • free - Exibe o uso de memória
  • ifconfig - Configura interfaces de rede
  • ip - Exibe/Configura interfaces de rede
  • ping - Testa a conectividade com outro host
  • netstat - Exibe conexões de rede
  • ss - Exibe conexões de rede (substituto do netstat)
  • scp - Copia arquivos via SSH
  • rsync - Sincroniza arquivos e diretórios
  • wget - Faz download de arquivos da web
  • curl - Faz requisições HTTP
  • history - Mostra o histórico de comandos
  • alias - Cria atalhos para comandos
  • unalias - Remove atalhos para comandos
  • uptime - Mostra há quanto tempo o sistema está em execução
  • whoami - Mostra o usuário atual
  • id - Mostra o ID do usuário
  • groups - Mostra os grupos do usuário
  • passwd - Altera a senha do usuário
  • su - Troca de usuário
  • sudo - Executa comandos como superusuário
  • exit - Sai do terminal ou de uma sessão
  • shutdown - Desliga o sistema
  • reboot - Reinicia o sistema
  • service - Gerencia serviços do sistema
  • systemctl - Controla o sistema e serviços
  • journalctl - Visualiza logs do sistema
  • dmesg - Mostra mensagens do kernel
  • lsblk - Lista dispositivos de bloco
  • mount - Monta sistemas de arquivos
  • umount - Desmonta sistemas de arquivos
  • fdisk - Gerencia partições de disco
  • mkfs - Cria sistemas de arquivos
